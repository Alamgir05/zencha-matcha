# Zencha Matcha 🍵

A full-stack e-commerce web app for matcha products.

## Features
- User authentication (JWT)
- Add to cart (MongoDB)
- Product listing page
- Admin product management

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB Atlas

## Setup

### Backend
cd backend  
cp .env.example .env  
npm install  
npm run dev  

### Frontend
npm install  
npm run dev
